v8cgi - CGI wrapper around V8 JS engine
=======================================

!!! EXTREME ALPHA STAGE - YOU HAVE BEEN WARNED !!!

1) Building

  a) download and build V8
  b) copy libv8.so to /usr/include/
  c) edit Makefile & run make (libmysqlclient-dev needed)
